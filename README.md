## Features
- Fixed header that changes on scroll
- Mobile-friendly hamburger menu
- Smooth scrolling and active link highlight
- Semantic HTML and accessibility basics
- Responsive layout (desktop & mobile)

## Tech
HTML5, CSS3, Vanilla JavaScript

## How to run locally
1. Clone: git clone https://github.com/shivamcodez01/PRODIGY_WD_01.git
2. `cd PRODIGY_WD_01`
3. Open `index.html` in your browser or run a simple static server.

## Deployment
Publish using GitHub Pages:
1. Push repo to GitHub as `PRODIGY_WD_01`.
2. Settings → Pages → Select `main` branch, root folder → Save.
3. Visit ` after deployment.
